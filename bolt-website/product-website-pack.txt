# Product Website Prompt for bolt.new

You are a web product team. Build a **modern product website** for **LuceData**, a commercial desktop app that manages SQL databases with integrated AI assistance.  

## PURPOSE
Create a marketing and documentation website that presents the product clearly, attracts users, and provides a beta registration form to receive download links for Windows/macOS installers. The app is currently **free in beta** — users register their interest and receive download access.  

## BRAND & STYLE
- Professional, clean, dark+light themes (like VS Code / modern dev tools).  
- Tech-savvy audience (developers, DBAs, data engineers).  
- Visual identity: screenshots of the app (connections tree, query editor, AI chat panel, results grid, schema explorer).  
- Tone: concise, confident, developer-friendly.  

## STRUCTURE
### 1. Landing Page
- Hero section with **headline**: “The AI-Powered SQL Desktop App”  
- Subtext: “Manage SQL Server, PostgreSQL, SQLite (Oracle and MySQL coming soon) with a modern cross-platform tool that combines query editing, results visualization, and AI-assisted SQL generation.”  
- Call-to-action: “Download for macOS · Windows · Linux” (buttons).  
- Screenshot carousel: show editor with results, chat with AI, schema explorer.  

### 2. Features
- **BYOM (Bring Your Own Model)**: In beta, connect your own AI model — OpenAI, Azure OpenAI, Gemini, Claude, Ollama local, and more. *Commercial version will include a custom, curated database-specialized model updated weekly.*  
- **Database Management**: Connect to SQL Server, PostgreSQL, SQLite today. Oracle and MySQL coming soon.  
- **Intelligent Explorer**: Navigate servers, schemas, tables, columns, indexes.  
- **SQL Editor**: Monaco-based editor with linting, formatting, split views.  
- **Results Grid**: Resizable grid with export to CSV.  
- **AI Chat Assistant**: Natural language to SQL with guardrails, schema awareness, conversation history. **AI proposes queries; you stay in control and execute them.**  
- **Full Write Access with Safeguards**: Execute DDL and DML operations — create tables, modify schemas, insert/update data. All operations require your confirmation and respect your connection's configured permissions.  
- **Secure by Design**: Confirmations for write operations, schema-aware guardrails, secrets stored safely.  
- **Cross-Platform**: macOS and Windows with auto-updates. Linux coming soon.  

### 3. Screenshots / Demo
- Showcase screenshots:  
  - Connections tree (servers, databases, schemas).  
  - Editor with queries + results grid.  
  - AI Assistant panel analyzing SQL.  
  - Schema explorer (tables, columns, keys, constraints).  

### 4. Beta Registration & Downloads
- **Registration Form**: Collect email, name, and intended use case.  
- Upon registration, users receive a confirmation email with download links for:  
  - macOS (.dmg)  
  - Windows (.msi)  
- Include checksums + release notes links in the email.  
- Note: Linux support coming in a future release.  

### 5. Documentation
- Link to `/docs`: Complete documentation portal with:
  - **Getting Started**: [Installation Guide](docs/installation.md)
  - **Configuration**: [Add AI Engine](docs/add-ai-engine.md), [Add Connection](docs/add-connection.md)
  - **Using LuceData**: [Connections Tree](docs/connections-tree.md), [Work Area](docs/work-area.md)
  - **Legal**: [Privacy Policy](privacy-policy.md), [Terms of Use](terms-of-use.md)
- Documentation home: [docs/README.md](docs/README.md)  

### 6. Roadmap & Updates
- Section: “Coming soon: Oracle, MySQL, team features.”  
- Changelog feed or blog for version releases.  

### 7. Footer
- Links: 
  - **Docs** ([/docs](docs/README.md))
  - **Support** (support@lucedata.com)
  - **Privacy Policy** ([/privacy](privacy-policy.md))

- Build with **Next.js** (or Astro) + Tailwind CSS.  
- Responsive and fast, SEO-optimized.  
- Deployable to Vercel (preferred).  
- Pages: `/`, `/downloads`, `/docs`, `/changelog`.

## Features Section

### 🌍 BYOM — Bring Your Own Model
**Title:** Your AI, your choice  
**Description:** In beta, connect your own AI model — OpenAI, Azure OpenAI, Anthropic Claude, Google Gemini, or even local Ollama. Configure multiple engines and switch instantly. *Coming in the commercial version: a custom, database-specialized model curated and updated weekly.*  

### 🗄️ Multiple Databases
**Title:** One client for many databases  
**Description:** SQL Server, PostgreSQL, and SQLite supported today. Oracle and MySQL are on the way.  

### 🌳 Visual Explorer
**Title:** Browse your schema visually  
**Description:** Servers, schemas, tables, columns, keys, triggers, and indexes presented in a modern tree view.  

### ✍️ Smart SQL Editor
**Title:** Code faster, with confidence  
**Description:** Monaco-based editor with formatting, linting, autocomplete, and split views.  

### 📊 Results Grid
**Title:** Analyze your data easily  
**Description:** Resizable, filterable results with one-click CSV export.  

### 🤖 AI Assistant
**Title:** Ask in plain English, get SQL back — you're in control  
**Description:** Natural language requests become SQL queries, grounded in your schema with guardrails to ensure safety. **The AI proposes, you approve and execute.** Every query runs with the permissions you've configured for each connection.  

### 🔒 Secure by Design
**Title:** Full power, with safeguards  
**Description:** Execute DDL (CREATE, ALTER, DROP) and DML (INSERT, UPDATE, DELETE) operations — all under your control. Every write operation requires confirmation, and all actions respect the permissions you've set for each database connection. Secure key storage keeps your credentials safe.  

### 💻 Cross-Platform
**Title:** Built for every developer  
**Description:** Native installers for macOS and Windows with automatic updates. Linux support coming soon.
