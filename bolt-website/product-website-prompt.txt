You are a web product team. Build a **modern product website** for a commercial desktop app that manages SQL databases with integrated AI assistance.  

## PURPOSE
Create a marketing and documentation website that presents the product clearly, attracts users, and provides a beta registration form to receive download links for Windows/macOS installers. The app is currently **free in beta** — users register their interest and receive download access.  

## BRAND & STYLE
- Professional, clean, dark+light themes (like VS Code / modern dev tools).  
- Tech-savvy audience (developers, DBAs, data engineers).  
- Visual identity: screenshots of the app (connections tree, query editor, AI chat panel, results grid, schema explorer).  
- Tone: concise, confident, developer-friendly.  

## STRUCTURE
### 1. Landing Page
- Hero section with **headline**: “The AI-Powered SQL Desktop App”  
- Subtext: “Manage SQL Server (and more soon) with a modern cross-platform tool that combines query editing, results visualization, and AI-assisted SQL generation.”  
- Call-to-action: “Download for macOS · Windows · Linux” (buttons).  
- Screenshot carousel: show editor with results, chat with AI, schema explorer.  

### 2. Features
- **BYOM (Bring Your Own Model)**: In beta, connect your own AI model — OpenAI, Azure OpenAI, Gemini, Claude, Ollama local, and more. *Commercial version will include a custom, curated database-specialized model updated weekly.*  
- **Database Management**: Connect to SQL Server, PostgreSQL, SQLite today. Oracle and MySQL coming soon.  
- **Intelligent Explorer**: Navigate servers, schemas, tables, columns, indexes.  
- **SQL Editor**: Monaco-based editor with linting, formatting, split views.  
- **Results Grid**: Resizable grid with export to CSV.  
- **AI Chat Assistant**: Natural language to SQL with guardrails, schema awareness, conversation history. **AI proposes queries; you stay in control and execute them.**  
- **Full Write Access with Safeguards**: Execute DDL and DML operations — create tables, modify schemas, insert/update data. All operations require your confirmation and respect your connection's configured permissions.  
- **Secure by Design**: Confirmations for write operations, schema-aware guardrails, secrets stored safely.  
- **Cross-Platform**: macOS and Windows with auto-updates. Linux coming soon.  

### 3. Screenshots / Demo
- Showcase real screenshots (provided) in a gallery with captions:
  - Connections tree (servers, databases, schemas).  
  - Editor with queries + results grid.  
  - AI Assistant panel analyzing/rewriting SQL.  
  - Schema explorer with columns, keys, constraints.  

### 4. Beta Registration & Downloads
- **Registration Form**: Collect email, name, and intended use case.  
- Upon registration, users receive a confirmation email with download links for:  
  - macOS (.dmg)  
  - Windows (.msi)  
- Include checksums + release notes links in the email.  
- Note: Linux support coming in a future release.  

### 5. Documentation
- Link to `/docs`: installation, quick start, configuring AI engines, guardrails, settings.  

### 6. Roadmap & Updates
- Section: “Coming soon: PostgreSQL, Google Gemini integration, migration support.”  
- Changelog feed or blog for version releases.  

### 7. Footer
- Links: Docs, GitHub (if public), Support, Privacy, Terms.  
- Copyright notice.  

## TECH
- Build with **Next.js** (or Astro) + Tailwind CSS.  
- Responsive and fast, SEO-optimized.  
- Deployable to Vercel (preferred).  
- Pages: `/`, `/downloads`, `/docs`, `/changelog`.  

## ACCEPTANCE CRITERIA
- Professional landing with product screenshots and feature highlights.  
- Clear CTAs to download per OS.  
- Features list well-structured with icons.  
- Docs and changelog pages accessible.  
- SEO tags (title, description, social share images).  
- Dark/light mode toggle.
